# BranchQ QA 체크리스트

## 사용법
- 각 항목을 직접 브라우저에서 테스트하고 `[x]` 체크
- 문제 발견 시 비고란에 증상 기록
- 테스트 환경: Chrome 최신 + localhost:5173

## D. 모달 (Modal)

### D1. M1 모델 선택
- [ ] GPT-4 버튼 클릭 → 모달 열림
- [ ] 라디오 3개 (GPT-4 추천, Claude 3.5, Gemini Pro)
- [ ] "추천" 배지 표시
- [ ] 선택 변경 → border accent + bg accent-light
- [ ] "적용" → 모달 닫기 + 헤더 모델명 변경
- [ ] "취소" → 모달 닫기 + 변경 없음

### D2. M3 공유
- [ ] ShareRow "공유" 버튼 → 모달 열림
- [ ] 4개 옵션: 링크 복사, 카카오톡, 이메일, 다운로드

### D3. M5 이메일
- [ ] 받는 사람, 제목, 내용 입력 필드

### D4. M6 확인
- [ ] 이상거래 "신고하기" → 확인 모달
