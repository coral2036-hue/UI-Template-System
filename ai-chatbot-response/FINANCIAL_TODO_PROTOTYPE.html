👤 사용자 인터페이스 흐름

일반 작업 (Task) 추가

  1. "💰 자금 To-Do" 탭 클릭 → SubPanel 열기
  2. "+ 추가" 버튼 클릭 → FinancialTodoModal 오픈
  3. Step 1: "📋 일반 작업" 선택
  4. Step 2: 제목, 카테고리, 마감일, 우선순위, 설명 입력
  5. 선택사항: 알림 설정 (시간, 반복 주기)
  6. "저장" 클릭 → 대시보드에 추가

질문 To-Do 추가

  1. "💰 자금 To-Do" 탭 클릭 → SubPanel 열기
  2. "+ 추가" 버튼 클릭 → FinancialTodoModal 오픈
  3. Step 1: "❓ 질문/분석" 선택
  4. Step 2: 질문 내용, 실행 예정일 입력
  5. 선택사항:
    • 자동 생성: ON (시간, 반복 주기 설정) → 정해진 시간에 자동 생성
    • 자동 생성: OFF → 대시보드에서 수동으로 [생성하기]
  6. "저장" 클릭 → 대시보드 질문 섹션에 추가

자동 답변 생성 흐름

  1. 페이지 로드 시 스케줄러 시작 (60초마다 체크)
  2. 현재 시간이 예정된 시간에 도달하면 감지
  3. 해당 질문에 대해 AI 서비스 호출
  4. 답변 생성 중: "⏳ 생성 중" 상태 표시
  5. 완료 또는 오류: "✅ 완료" 또는 "❌ 오류" 상태 표시
  6. [전체 보기] 클릭 시 QuestionAnswerModal에서 전체 답변 확인
