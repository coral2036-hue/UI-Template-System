# 뱅킹 화면 UI 가이드
> node-id: 9422-71342 | update_2025.05.14

---

## 1. 화면 구성

| 섹션 | 하위 화면 |
|------|-----------|
| 적금/적립식 | 적금관련조회 / 만기관리 / 적립식금융상품 |
| 대출 | 대출관련 조회 |
| 외환 | 외환 관련 화면 |
| 공과금 | 공과금 납부 화면 |
| 이원매/연금 | 이원매 관련 다수 화면 |
| 기타 | 기타 뱅킹 서비스 |

---

## 2. 공통 디자인 규칙

| 항목 | 값 |
|------|-----|
| 폰트 | 맑은 고딕 12px |
| 입력/버튼 높이 | 28px |
| 강조 버튼 | 파란색 솔리드 |
| 라인 버튼 | 파란색 라인 |
| 서브 강조색 | 주황색 (만기/주의 상태) |

---

## 3. 화면별 컴포넌트 구성

### 3-1. 적금관련조회
- 조회 조건: 계좌번호(Select) / 기간(DatePicker~DatePicker)
- 그리드: 계좌번호 / 적금종류 / 납입금액 / 만기일 / 상태
- 만기 임박 행: 주황색 강조

### 3-2. 공과금 납부
- 입력: 납부기관(Select) / 납부번호(Input) / 납부금액(Input)
- 버튼: [취소](라인) / [납부실행](강조)

---

## 4. Figma 프롬프팅 예시

```
WIN-CMS 적금관련조회 화면을 생성해줘.
- 조회 조건: * 계좌번호(Select), * 기간(DatePicker~DatePicker)
- 버튼: [초기화](라인), [조회](강조 솔리드) — 우측 하단
- 그리드: 계좌번호 / 적금종류 / 납입금액 / 만기일 / 상태 (5컬럼)
- 만기 임박 행 주황색 강조 표시
```
