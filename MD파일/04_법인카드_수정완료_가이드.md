## 2. 공통 디자인 규칙

| 항목 | 값 |
|------|-----|
| 폰트 | 맑은 고딕 12px |
| 입력/버튼 높이 | 28px |
| 강조 버튼 | 파란색 솔리드 |
| 데이터 분석 차트 | 바 차트 / 라인 차트 조합 |

---

## 3. 화면별 컴포넌트 구성

### 3-1. 카드사용데이터분석
- 상단: 기간 필터 + 부서/카드 Select
- 본문: 차트 영역 + 요약 수치
- 하단: 상세 그리드

### 3-2. 카드사용내역조회
- 조회 조건: 카드번호(Select) / 기간(DatePicker) / 사용구분(Select)
- 그리드: 사용일시 / 가맹점명 / 금액 / 승인번호 / 상태
- 버튼: [초기화](라인) / [조회](강조) / [엑셀다운](라인)
