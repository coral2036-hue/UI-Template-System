# 기타 화면 UI 가이드
> node-id: 13765-2498

---

## 1. 화면 구성

| 섹션 | 하위 화면 |
|------|-----------|
| WIN-CMS 메인 대시보드 | 메인 홈 / 기능 바로가기 / 공지사항 |
| 기능 바로가기 | 아이콘 메뉴 |
| 이미지 전달 | 개발팀 전달용 화면 이미지 모음 |

---

## 2. 공통 디자인 규칙

| 항목 | 값 |
|------|-----|
| 폰트 | 맑은 고딕 12px |
| 헤더 | WIN-CMS 로고 + 상단 네비게이션 |
| 아이콘 메뉴 | 아이콘 + 텍스트 레이블 조합 |

---

## 3. 화면별 컴포넌트 구성

### 3-1. WIN-CMS 메인 대시보드
- 상단: WIN-CMS 로고 + 사용자명 + 로그아웃 버튼
- 중단: 기능 바로가기 아이콘 메뉴 그리드 (2~3열)
- 하단: 공지사항 목록 / 최근 이체 내역 요약

### 3-2. 이미지 전달 섹션
- 격자형 썸네일 배치
- 각 이미지에 화면명 레이블 표시

---

WIN-CMS 메인 대시보드 화면을 생성해줘.
