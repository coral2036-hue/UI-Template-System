# 부가서비스 화면 UI 가이드
> node-id: 13896-61570 | update_2025.05.15

---

## 1. 화면 구성

| 섹션 | 하위 화면 |
|------|-----------|
| 전자세금계산서(택스빌365) | 세금계산서 발행 / 수정세금계산서 발행 |
| RCMS | RCMS 환경설정 (01.21 추가) |

---

## 2. 공통 디자인 규칙

| 항목 | 값 |
|------|-----|
| 폰트 | 맑은 고딕 12px |
| 입력/버튼 높이 | 28px |
| 강조 버튼 | 파란색 솔리드 |
| 환경설정 배지 | 주황색 (신규 추가 표시) |

---

## 3. 화면별 컴포넌트 구성

### 3-1. 전자세금계산서 발행
- 입력: 공급자 정보 / 공급받는자 정보 / 품목 정보
- 필수값: 사업자번호 / 공급가액 / 세액
- 버튼: [취소](라인) / [임시저장](라인) / [발행](강조)

### 3-2. RCMS 환경설정
- 배지: 주황색 "01.21 추가"
- 레이아웃: 좌측 메뉴 + 우측 설정 폼
- 버튼: [초기화](라인) / [저장](강조)

---

```
WIN-CMS 전자세금계산서 발행 화면을 생성해줘.

```
