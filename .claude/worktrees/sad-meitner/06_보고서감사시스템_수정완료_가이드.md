## 1. 화면 구성

| 섹션 | 하위 화면 |
|------|-----------|
| 운영보고서 > 보고서종합현황 | 운영모니터 / 기업대상현황 / 원카드매니저현황 |
| 운영보고서 > WIN-CARD Manager 보고서 | 관리자보고서요약 |
| 감사시스템 | 주시데이터현황 |
| 감사시스템 > 자금관리모니터링 | 자금관리모니터링 (2종) |
| 감사시스템 > 스크래핑법인카드모니터링 | 정보1 / 정보2 |
| 감사시스템 > WIN-CARD Manager 모니터링 | 모니터링현황조회 |

## 2. 공통 디자인 규칙

| 항목 | 값 |
|------|-----|
| 폰트 | 맑은 고딕 12px |
| 입력/버튼 높이 | 28px |
| 모니터링 상태 | 정상(파란) / 경고(주황) / 오류(빨간) |

---

## 3. 화면별 컴포넌트 구성

### 3-1. 보고서종합현황
- 상단: 기간 필터 + 유형 Select
- 본문: 요약 수치 + 차트 영역
- 하단: 상세 그리드

### 3-2. 자금관리모니터링
- 상태 인디케이터: 정상/경고/오류 색상 구분

### 3-3. 스크래핑법인카드모니터링
- 수집 성공/실패 상태 배지
