# 전자결제 화면 UI 가이드
> node-id: 13726-301509 | update_2025.05.15

---

## 1. 화면 구성

| 섹션 | 하위 화면 |
|------|-----------|
| 판매기업 > B2B(협력) | B2B(협력) / 대금결제/담당자처리 등 |
| 판매기업 > 상생파트너론(협력) | 쟁점조건서대출건신 |
| 구매기업 > B2B(주계약) | 대출처리목록 / 전자달발행책처리 |
| 타기관전자결제 | 전자정산상승조회(통합) |
| 전자이음/채권 | 기금관리이음 발행 / 전자매출채권 발행승인 |

---

## 2. 공통 디자인 규칙

| 항목 | 값 |
|------|-----|
| 폰트 | 맑은 고딕 12px |
| 입력/버튼 높이 | 28px |
| 강조 버튼 | 파란색 솔리드 |
| 신규/주기 배지 | 주황색 |

---

## 3. 화면별 컴포넌트 구성

### 3-1. 판매기업 B2B
- 조회 조건: 기간(DatePicker) / 거래처(Select) / 상태(Select)
- 그리드: 결제일 / 거래처명 / 결제금액 / 상태 / 담당자
- 담당자처리 팝업: 중형(L)

### 3-2. 전자이음/채권
- 프로세스: 발행요청 → 담당자확인 → 승인
- 주기 배지: 주황색 "(주기)" 표시
