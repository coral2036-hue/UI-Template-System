## 3. 화면별 컴포넌트 구성

### 3-1. 지급공통 팝업
- (팝)자주쓰는계좌: 중형(L), 계좌 목록 그리드
- (팝)거래처계좌: 중형(L), 거래처+계좌 선택
- (팝)이체결과조회: 대형(XL), 결과 그리드

### 3-2. 전별이체
- 입력: 출금계좌(Select) / 수취인명(Input) / 이체금액(Input) / 이체일(DatePicker)
- 필수값: 출금계좌, 수취인명, 이체금액
- 버튼: [취소](라인) / [이체실행](강조)

### 3-3. 집금결과조회
- 헤더 배지: 주황색 강조
- 집금 성공/실패 상태별 컬럼 구성
