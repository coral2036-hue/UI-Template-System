# 설정 화면 UI 가이드
> node-id: 10125-2 | update_2025.05.15

---

## 1. 화면 구성

| 섹션 | 하위 화면 |
|------|-----------|
| 등록관리 > 초기설정 | 초기설정 관련 다수 |
| 통보관리 | 통보 설정 화면 |
| 일정스케줄관리 | 스케줄 등록/조회/관리 |
| 시스템관리 | 시스템 환경 설정 |
| 정책관리 | 정책 관련 관리 |
| 로그/규범정책관리 | 로그 조회 및 규범 정책 |

---

## 2. 공통 디자인 규칙

| 항목 | 값 |
|------|-----|
| 폰트 | 맑은 고딕 12px |
| 입력/버튼 높이 | 28px |
| 설정 레이아웃 | 좌측 트리 메뉴 + 우측 설정 폼 |

---

## 3. 화면별 컴포넌트 구성

### 3-1. 일정스케줄관리
- 캘린더 뷰 + 목록 뷰 전환
- 등록 팝업: 중형(L)
- 버튼: [취소](라인) / [등록](강조)

### 3-2. 시스템관리
- 좌측 트리 메뉴 + 우측 상세 폼

### 3-3. 로그조회
- 기간 DatePicker + 유형 Select
- 그리드: 일시 / 사용자 / 작업 / IP / 결과

---

## 4. Figma 프롬프팅 예시

```
WIN-CMS 일정스케줄관리 화면을 생성해줘.
- 상단: 월 네비게이션 + [목록보기](라인) 버튼
- 본문: 캘린더 뷰 (7컬럼 주간 그리드)
- 등록 팝업: 중형(L)
- 버튼: [취소](라인), [등록](강조) — 우측 하단
```
